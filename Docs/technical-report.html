Abstract

A mash up which recommends various resources and they are available in the form of RSS feed

Introduction

This application is a mash-up web application and a browser extension which recommends various resources of interest for users. TOPIKS is a content aggregator designed to serve its clients with a vast collection of resources, from news and documents to music, videos and even photography. Recommendations are reported with real time notification and they are available in the form of RSS feed.

Technologies used

HyperText Markup Language(HTML) is the most basic language for web application. The latest version of HTML, which it is HTML5 allows to build more diverse and powerful web applications and websites with new elements, new attributes and new behaviors.

Cascading Style Sheets (CSS) is style sheet language. It is use in web development for to beautify your website or web application

Vanilla JavaScript (JS) is used for both backend and frontend. In backend development is used to create the server and to do to make the connection between client, server and database. In frontend development it is used to create things that do something like a button.

Mongoose is a library for MongoDB, which is a no relational database, and it is base on NoSQL.

AJAX (short for "Asynchronous JavaScript and XML")

Implementation

We used HTML5, JavaScript and CSS for frontend to create a web page.

We used Node.js (JavaScript) for backend to make the connection between frontend and database and server.

For storage we used MongoDB and library mongoose for database operations.
